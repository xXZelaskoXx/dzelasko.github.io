Show Me the Money: Currency Converter

Welcome to Show Me the Money, an online tool for currency conversion. To use this tool, simply select from the dropdown menus the currencies you would like to convert as well as the amount of the first currency to be converted. For a conversion rate, enter 1 in the currency amount box. Additionally, the tool will list games that the amount provided would be able to purchase per the Cheapshark API should the funds be converted to USD.

Thank you for using SMTM!
